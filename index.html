What is Screen size and Viewport?

Screen Size: It refers to the actual physical dimensions of a display screen, typically measured diagonally. It represents the entire surface area of the screen, including any borders or edges.

Viewport: The viewport is the visible portion of a web page that is displayed within the browser window. It's the part of the web page that users can see without scrolling. The size of the viewport can vary depending on the device's screen size, resolution, and browser settings. In web design, adapting content to fit within the viewport ensures optimal presentation on different devices.
